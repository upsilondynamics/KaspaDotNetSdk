# KaspaDotNetSdk

A lightweight C# / .NET SDK for interacting with the [Kaspa](https://kaspa.org) blockchain using gRPC.

This SDK simplifies connecting to a Kaspa node, retrieving network info, getting block data, querying balances, and submitting transactions â€” all through a familiar .NET API surface.

## Features

- Connect to a Kaspa node via gRPC
- Get network info and block count
- Retrieve peer info
- Strongly typed models
- Async/await support
- Designed for extensibility
